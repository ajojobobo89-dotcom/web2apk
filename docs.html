📚 Dokumentasi LIFX Web2APK

Selamat datang di dokumentasi resmi LIFX Web2APK. Platform ini memungkinkan Anda mengubah kode HTML, website, atau file ZIP menjadi aplikasi Android (APK) yang nyata dan siap diinstal.

Versi 13.0 - Terakhir diperbarui: 6 Maret 2026

🚀 Quick Start

Ikuti 3 langkah mudah berikut untuk membuat APK pertama Anda:

1

Siapkan HTML

Pastikan HTML Anda sudah responsive untuk tampilan mobile. Gunakan viewport meta tag:

<meta name="viewport" content="width=device-width, initial-scale=1">
2

Upload ke Builder

Buka halaman Builder, pilih metode input (HTML/URL/ZIP), dan masukkan source code Anda.

3

Generate APK

Klik tombol "GENERATE APK", tunggu 30-60 detik, dan download APK Anda.

📄 Input HTML

Anda dapat langsung menempelkan kode HTML ke dalam text area. Berikut contoh struktur HTML yang baik:

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>My App</title>
    <style>
        body {
            font-family: Arial, sans-serif;
            margin: 0;
            padding: 20px;
            background: #f5f5f5;
        }
    </style>
</head>
<body>
    <h1>Hello World!</h1>
    <p>Welcome to my app</p>
    <script>
        console.log('App loaded');
    </script>
</body>
</html>
Tip: Gunakan CSS inline atau internal untuk hasil terbaik. Hindari link ke file eksternal jika tidak diperlukan.

🔗 Input URL

Jika website Anda sudah online, cukup masukkan URL-nya. Sistem akan mendownload seluruh h beserta asetnya.

Contoh URL yang valid:

  • https://example.com
  • https://myapp.netlify.app
  • https://localhost:3000 (untuk development)
Catatan: Pastikan website Anda dapat diakses publik. Jika menggunakan localhost, gunakan tunneling seperti ngrok.

📦 Input ZIP

Upload file ZIP yang berisi seluruh file website Anda. Struktur folder yang direkomendasikan:

my-app.zip
├── index.html
├── css/
│   └── style.css
├── js/
│   └── script.js
└── images/
    ├── logo.png
    └── icon.png

Pastikan file utama bernama index.html dan berada di root folder.

🔌 API Reference

LIFX Web2APK menyediakan REST API untuk integrasi dengan aplikasi lain.

Base URL

https://apk.lifx.com/api

Authentication

Semua request ke API memerlukan API key yang dikirim via header:

X-API-Key: your-api-key-here

POST /build

Endpoint untuk membuat APK baru.

Request (multipart/form-data)

Field Type Required Description
action string yes build_html / build_url / build_zip
app_name string yes Nama aplikasi
package_name string yes Package name (com.example.app)
html string for action=build_html HTML content
